Imputing missing values

Calculate number of NA values from steps in dataframe data

sum(is.na(data$steps)==TRUE)

## [1] 2304

Strategy for imputing missing values in original dataset is as follows. It is to replace NAs by average steps of the interval when NA value is recorded. Processing data to calculate the average steps taken for each day. It is preparation for the inputing missing values.

Create temporary vector filling. It contains steps with NAs replaced with average steps for its interval. When the element of filling is NA, first, find the interval of NA value is recorded by match() function. And replace NA value by steps averaged by interval which is calculated former section.

filling <- data$steps
for(i in 1:length(filling)){
    if(is.na(filling[i])){
        index <- match(data[i,"interval"],intervalaverage$interval)
        filling[i] <- intervalaverage[index,"steps"]
    }
}

Create new dataset newdf with replaced NAs. And rename colnames.

newdf <- cbind(filling,data[2:3])
names(newdf) <- c("steps","date","interval")

Making a histogram of new dataset, calculating mean and median. These procedures are same as that for original dataset.

newdaytotal <- aggregate(steps ~ date,newdf,sum)

hist(newdaytotal$steps,xlab="Total number of steps taken per day",main="")

Are there differences in activity patterns between weekdays and weekends?

Dataframe weekendlist contains pairs of the day of the week and whether it is weekday or weekend.

weekendlist <- data.frame(day=c("Monday","Tuesday","Wednesday","Thursday","Friday","Saturday","Sunday"),
                          we=c(rep("Weekday",5),rep("Weekend",2)))

New vector wd contains whether the day is weekday or weekend. First, created as the copy of date of original dataset. And converted into the day (Sunday to Saturday).

wd <- data$date
wd <- weekdays(as.Date(wd))

For all element of wd, find whether the element is weekday or weekend by matching weekendlist. And replace the day by weekday or weekend.

for(i in 1:length(wd)){
  index <- match(wd[i],weekendlist$day)
  wd[i] <- as.character(weekendlist[index,"we"])

}
wd <- as.factor(wd)

Final dataframe n2df is combination of NA replaced dataframe and the vector indicates the day is weekday or weekend. stepsintwd is the dataframe contains col1=interval, col2 = weekday/weekend, col3 = average steps in each interval and weekday/weekend.

n2df <- cbind(newdf,wd)
stepsintwd <- aggregate(steps ~ interval + wd,n2df,mean)

Two time series plots which show weekday and weekend average activity patterns averaged across all the days. To plot panel plots, ggplot2 package is used.

g <- ggplot(stepsintwd,aes(interval,steps))
g + geom_line() + facet_grid(wd ~ .) + labs(y="Average steps taken")
